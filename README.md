# Sales_performance_dashboard
This Power BI dashboard provides a comprehensive overview of store sales performance by combining sales, profit, customer, and shipping metrics into a single interactive report. It enables users to monitor business performance, compare trends across years, and identify high-performing regions, product categories, and customer segments.
